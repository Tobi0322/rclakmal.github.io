Hi. I'm Lakmal Padmakumara.

I'm a programmer, an engineer and a science enthusiastic from Sri Lanka, currently based in Munich, Germany following my Masters in Computer Science at Technical University of Munich. My favourite programming languages so far are Python, Java and Javascript (in that order). I'm currently employed as a working student at CLIQZ, a munich tech startup with the dream of making a smarter search & better user-agents for the web.
